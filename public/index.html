Max-Heap Architecture

We utilize a Binary Heap data structure to manage the priority queue. This ensures that the retrieval of the highest priority patient takes only O(log n) time, making the system efficient even with thousands of concurrent patients.

  • Critical > High > Medium > Low
  • FCFS fallback for equal severity
  • O(1) Peek / O(log n) Extract
class PriorityQueue {
  constructor() { ... }

  // Time: O(log n)
  enqueue(patient) {
    this.heap.push(patient);
    this.bubbleUp();
  }

  // Time: O(log n)
  dequeue() {
    return this.heap.shift();
  }
}
